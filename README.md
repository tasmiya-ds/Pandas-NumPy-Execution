
## Pandas & NumPy Execution
This task focuses on data analysis using Pandas and NumPy on a CSV dataset containing student performance data.

### Concepts Covered
- Loading and exploring CSV datasets
- Displaying first and last rows of data
- Creating calculated columns
- Filtering data using multiple conditions
- Grouping data and performing aggregation (mean and count)

### Files
- `task2_pandas_numpy.ipynb` – Jupyter Notebook containing analysis
- `Cleaned_Student_Performance.csv` – Dataset used for analysis

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook
